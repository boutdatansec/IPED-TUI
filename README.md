# IPED Digital Forensic Tool

IPED is an open source software that can be used to process and analyze digital evidence, often seized at crime scenes by law enforcement or in a corporate investigation by private examiners.

This fork is an unofficial Terminal User Interface (TUI) of IPED. Check the original [IPED project](https://github.com/sepinf-inc/IPED) by **[@lfcnassif](https://github.com/lfcnassif)** to check the features and usage of IPED.

The main purpouse is to allow the processing on headless servers with a little bit of feedback from GUI. And maybe, _just maybe_, some basic features of analysis GUI.
